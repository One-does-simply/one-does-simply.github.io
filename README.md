# one-does-simply.github.io

The org-level landing page for [One Does Simply](https://github.com/One-does-simply).
Deployed automatically to <https://one-does-simply.github.io/> on every
push to `main`.

Content is a single [index.html](index.html) introducing ODS and
listing the active + planned families. When a new family lands,
update the `.families` section.
